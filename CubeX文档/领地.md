# 领地系统

你想有一块自己的地盘吗？在 CubeX，你可以圈地为王。别人不能破坏你的领地，你说了算。

> 主指令 `/lands`（简写 `/l`），直接输入打开管理界面。

---

## 创建自己的领地

### 第一步：取个名字

```
/lands create 领地名 标签
```

标签是 2-4 个字母的缩写，会在你名字旁边显示。

比如：
```
/lands create 青云镇 QY
```

### 第二步：圈地

站在你想占的地盘上：

```
/lands claim
```

你脚下那个区块就是你的了。多站几个地方多输几次，把领地圈大。

> 一个区块是 16x16 格。想看区块边界？按 `F3+G`（显示区块网格线）。

### 快速圈地技巧

| 我想... | 输入 |
|---------|------|
| 一次圈周围一大片 | `/lands claim radius 3`（半径几个区块就填几） |
| 边走边圈 | `/lands claim auto`（走完再输一次关闭） |
| 用木斧选区后圈 | 拿木斧左右键选区 → `/lands selection` → `/lands claim` |

## 怎么让朋友进来

在你的领地里，其他人默认什么都不能做。你可以：

### 信任朋友

```
/lands trust 玩家名
```

对方被信任后可以盖房子、开箱子（具体能做什么看他的角色）。

### 给朋友分配角色

角色决定了对方有多大权限：

| 角色 | 能做什么 |
|------|---------|
| Co-Owner（副领主） | 几乎和你一样，能管理领地 |
| Resident（居民） | 能盖房子、用箱子 |
| Visitor（访客） | 只能看，不能碰 |

用 GUI 改角色权限：打开 `/lands` → 角色 → 点一个角色 → 开关具体权限。

给玩家设角色：
```
/lands setrole 玩家名 * Co-Owner
```

`*` 表示在所有区域生效。

---

## 怎么不让陌生人进来

```
/lands ban 玩家名
```

被封的人进不了你的领地。

取消封禁：
```
/lands unban 玩家名
```

## 领地日常管理

打开 `/lands` GUI 可以搞定大部分事。以下是常用命令：

| 我想... | 输入 |
|---------|------|
| 看领地信息 | `/lands info` |
| 看周围领地地图 | `/lands map` |
| 传送到领地出生点 | `/lands spawn` |
| 设置出生点 | `/lands setspawn` |
| 看领地边界 | `/lands view`（边界会发光） |
| 改领地名字 | `/lands rename 新名字` |
| 存钱交维护费 | `/lands deposit 金额` |
| 看领地余额 | `/lands balance` |
| 退出领地 | `/lands leave` |
| 删除领地 | `/lands delete` |

> 领地每天有维护费，记得存钱。领地越大维护费越高。

## 子区域：领中领

子区域是领地内部的 3D 小区域。比如你在领地里有栋楼想单独租出去，就给这栋楼建个子区域。

### 创建子区域

1. 输入 `/lands` → 点 Areas → 点 Add area
2. 聊天栏输入子区域名字
3. 拿木斧，左键/右键选出你要的 3D 范围（两个对角）
4. 输入 `/lands selection assign 子区域名`

### 子区域管理

| 我想... | 输入 |
|---------|------|
| 打开子区域设置 | `/lands menu here` |
| 信任某人到子区域 | `/lands trust 玩家名 子区域名` |
| 取消信任 | `/lands untrust 玩家名 子区域名` |
| 删除子区域 | `/lands delete here` |

## 国家：多个领地一起玩

几个领地可以联合成一个国家。

### 创建国家

```
/nations create 国家名
```

你的领地自动成为首都。

### 邀请别人加入

```
/nations trust 对方领地名
```

对方会收到邀请，输入 `/nations accept 你的国家名` 就加入了。

### 国家常用命令

| 我想... | 输入 |
|---------|------|
| 打开国家界面 | `/nations menu` |
| 邀请领地加入 | `/nations trust 领地名` |
| 踢出领地 | `/nations untrust 领地名` |
| 退出国家 | `/nations leave` |
| 传到首都 | `/nations spawn` |
| 解散国家 | `/nations delete` |

## 出租和出售

你可以把子区域租出去或卖掉赚钱。

### 出租

站在子区域内，打开设置：

```
/lands menu here
```

点租赁设置，填入格式：`时间间隔 最大时长 费用`

玩家看到你放的告示牌，右键就能租。

### 租约管理

| 我想... | 输入 |
|---------|------|
| 看可租的领地 | `/lands rent list` |
| 退租（我是租客） | `/lands rent cancel` |
| 收回（我是房东） | `/lands rent remove` |

## 战争

领地之间可以打仗。别轻易宣战，会死人的。

### 宣战

```
/wars declare 对方领地名
```

### 战争相关命令

| 我想... | 输入 |
|---------|------|
| 打开战争界面 | `/wars menu` |
| 宣战 | `/wars declare 目标` |
| 参战传送 | `/wars spawn` |
| 看当前战争 | `/wars info` |

> 盟友可以参战帮你。

## 领地聊天

在聊天框里发消息时加个前缀：

```
@land 大家好啊
```

只有你领地内的人能看到。

---

## 小贴士

- 输入 `/lands map` 看看周围有没有别人的领地
- 领地越大维护费越高，记得 `/lands deposit` 存钱
- 卡住了输入 `/lands unstuck` 传送回安全位置
- 想去野外探险？`/lands wild` 随机传送
- 在多个领地时用 `/lands edit 领地名` 切换你要操作的目标
