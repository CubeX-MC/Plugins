# 铁路交通系统

CubeX 的铁路是由玩家自己建设的。你可以乘坐别人建好的线路。

---

## 怎么乘坐

1. 找到地上铺着的**红石铁轨**
2. 对着铁轨**右键**，矿车自动来
3. 矿车自己开，到站就停

下车后如果有换乘线路，找到对应停靠点的铁轨右键继续坐。

### 怎么知道有哪些线路

| 我想... | 输入 |
|---------|------|
| 看所有线路 | `/m line list` |
| 看某条线路经过哪些站 | `/m line stops 线路名` |
| 看某条线路多少钱 | `/m line priceinfo 线路名` |
| 看这个站能换乘到哪 | `/m stop listtransfers 站名` |

> 所有名字都用英文 ID，比如 `/m line stops red_line`。

### 常见问题

**右键铁轨没反应？**
检查线路状态，被暂停的线路不能坐。输入 `/m line info 线路名`。

**矿车开一半不动了？**
可能是铁轨断了或者路线没录全。联系线路主人检查 `/m line routeinfo 线路名`。

**怎么知道哪站能换乘？**
站在站台上输入 `/m stop listtransfers 站名`。

## 建造铁路

> 以下内容需要你持有**铁路宝石**才能操作。没有宝石只坐车的话，上面那节就够了。

### 修一条完整线路

以建一条叫 `red_line`（红线）的线路为例，全程七步。

#### 第一步：铺铁轨

从起点到终点，用铁轨把路铺好。必须是**连续的普通铁轨**（红石铁轨是停靠点，后面会设）。

#### 第二步：建站台，圈区域

在每个站点，用方块搭好站台。然后拿**木斧**，左键点一个角，右键点对角，把整个站台框起来。

#### 第三步：创建停靠区

```
/m stop create 站名 显示名称
```

比如：
```
/m stop create spawn_station 出生点站
/m stop create village_station 村庄站
```

每个站都要做一次。

#### 第四步：设置停靠点

站在站台里，对着**红石铁轨**输入：

```
/m stop setpoint 站名
```

比如：
```
/m stop setpoint spawn_station
```

> 不指定朝向角度的话，系统自动取你面朝的方向。

#### 第五步：创建线路

```
/m line create red_line 红线
```

#### 第六步：把车站加入线路

按列车行驶顺序，一站一站加：

```
/m line addstop red_line spawn_station
/m line addstop red_line village_station
```

> 不指定位置就自动加到末尾。想在中间插入就加数字：`/m line addstop red_line new_stop 2`

#### 第七步：录制路线

坐上一辆矿车（或自己放一辆），沿着铁轨从起点开到终点：

```
/m line recordroute red_line
```

再输一次结束录制。搞定了！现在对着站台上的红石铁轨右键就能坐车。

### 怎么设票价

```
/m line setprice 线路名 定价方式 [参数]
```

四种方式：

| 方式 | 示例 | 意思 |
|------|------|------|
| 固定价 | `/m line setprice red_line 10` | 每次 10 块 |
| 按距离 | `/m line setprice red_line distance 2 0.1 50` | 起步 2 块，每格 +0.1，封顶 50 |
| 按站数 | `/m line setprice red_line interval 2 1 20` | 起步 2 块，每站 +1，封顶 20 |
| 免费 | `/m line setprice red_line 0` | 不要钱 |

> 基价是起步费，乘数是每格/每站的加价，封顶是最高票价。

### 线路管理

| 我想... | 输入 |
|---------|------|
| 改线路名 | `/m line rename red_line 新红线` |
| 改线路颜色 | `/m line setcolor red_line &c` |
| 改最高速度 | `/m line setmaxspeed red_line 2` |
| 暂停运营 | `/m line setstatus red_line suspended` |
| 恢复运营 | `/m line setstatus red_line normal` |
| 保护铁轨不被破坏 | `/m line protect red_line on` |
| 关闭保护 | `/m line protect red_line off` |
| 创建反向线路 | `/m line clonereverse red_line blue_line` |
| 删除线路 | `/m line delete red_line` |

---

### 设置换乘

如果你的站可以换到别的线路：

```
/m stop addtransfer 你的站名 要换的线路名
```

比如在 `spawn_station` 可以换乘 `blue_line`：
```
/m stop addtransfer spawn_station blue_line
```

---

### 怎么让朋友一起管

| 我想... | 输入 |
|---------|------|
| 让朋友管线路 | `/m line trust 线路名 玩家名` |
| 收回管理权 | `/m line untrust 线路名 玩家名` |
| 把线路送给别人 | `/m line owner 线路名 玩家名` |
| 让朋友管车站 | `/m stop trust 站名 玩家名` |
| 收回车站管理权 | `/m stop untrust 站名 玩家名` |
