# 合同系统

合同系统让你和其他玩家签订正式协议。雇人干活、合伙做事、打赌下注，都可以用合同来保证公平。钱由服务器托管，谁也赖不掉。

> 主指令 `/contract`（简写 `/ct`），直接输入打开主界面。

---

## 三种合同

| 类型 | 干什么用的 | 谁出钱 |
|------|-----------|--------|
| 服务合同 | 你出钱雇人干活。对方干完你验收付款 | 雇主出 |
| 对赌合同 | 你和一个人打赌，各出一样的钱，找个裁判决定谁赢 | 双方各出一半 |
| 合伙合同 | 你和一个人合伙做事，各出各的钱，都同意才算完成 | 双方各自出 |

---

## 我想雇人干活（服务合同）

### 用指令创建

```
/contract service 报酬 期限小时 标题|描述
```

比如你出 $500 雇人帮你盖个房子，3 天内完成：

```
/contract service 500 72 雇人盖房|帮我在出生点旁边盖一栋橡木小屋，两层带阳台
```

> 期限最大 168 小时（7 天），最小 1 小时。72 小时就是 3 天。

创建时系统会立即从你的账户扣走：**报酬 + $20 创建费**。报酬存在服务器托管，谁也动不了。

### 用 GUI 创建

1. 输入 `/contract` 打开主界面
2. 点 **创建合同**（书本图标）
3. 选 **服务合同**
4. 按提示填写：报酬金额 → 期限 → 标题 → 描述
5. 确认无误后，在铁砧界面输入你的名字签名

签名后合同立即生效，进入合同大厅等待别人来接。

## 我想接活赚钱

### 找活

```
/contract list
```

列出所有公开的服务合同。翻页：`/contract list 2`

或者打开 GUI，点 **合同大厅**（绿宝石图标），浏览所有可接的合同。

### 接活

```
/contract accept 合同 ID
```

合同 ID 是一串短代码，在列表里能看到。也可以在 GUI 里点合同然后点 **接受合同**。

### 交活

活干完了，提交给雇主验收：

```
/contract submit 合同 ID
```

或者在 GUI 里打开你的合同，点 **提交完成**。

### 收钱

雇主验收通过后，钱自动到账。你会收到 `报酬 - 5% 手续费`，剩下的就是你的。

> 如果雇主拖着不验收，超过 72 小时系统自动通过。

## 我想跟人打赌（对赌合同）

找个对手，各出同样的钱，找个裁判来决定谁赢。

```
/contract wager 对手 赌注 期限小时 裁判 标题|描述
```

比如你和 Steve 各押 $200，找 Alex 当裁判，赌谁钓的鱼多：

```
/contract wager Steve 200 24 Alex 钓鱼大赛|一小时内谁钓的鱼多谁赢，Alex 做裁判
```

创建后 Steve 和 Alex 都会收到通知。Steve 接受后合同生效。等双方都准备好，Alex 输入：

```
/contract resolve 合同 ID a
```

`a` 代表你赢，`b` 代表 Steve 赢。赢家拿走全部（扣除手续费）。

## 我想跟人合伙（合伙合同）

两个人各出不同金额，合伙做一件事。双方都同意才算完成。

```
/contract partner 合伙人 你出的钱 对方出的钱 期限小时 标题|描述
```

比如你出 $500，对方出 $300，合伙开一家店：

```
/contract partner Steve 500 300 168 合伙开店|我出钱买材料，Steve 负责建店，利润平分
```

双方都确认后，合同完成，各自拿回出资金额。

## 合同管理

### 查看我的合同

```
/contract my
```

列出所有与你有关的合同（你创建的、你接的、你裁判的）。

### 查看合同详情

```
/contract info 合同 ID
```

### 取消合同

```
/contract cancel 合同 ID
```

取消规则：
- **没人接的合同**：雇主取消，全款退回
- **进行中的合同**：取消方可能有损失，由系统按规则退款
- **双方同意的合同**：协商一致后取消

### 发起争议

如果对方耍赖，你可以把合同标为争议状态：

```
/contract dispute 合同 ID 争议原因
```

管理员会介入处理。

## GUI 操作指南

输入 `/contract` 打开主界面，六个入口：

| 图标 | 功能 | 什么时候用 |
|------|------|-----------|
| 书与笔 | 创建合同 | 想雇人、打赌、合伙 |
| 钟（带数字） | 待办事项 | 有合同等你操作（验收、裁判等） |
| 绿宝石 | 合同大厅 | 浏览所有公开的合同找活干 |
| 书 | 我的合同 | 查看自己相关的所有合同 |
| 知识之书 | 帮助 | 在聊天栏显示帮助信息 |
| 工作台 | 管理员界面 | 仅管理员可见 |

### 签名确认

涉及金钱的操作都需要签名确认。在铁砧界面输入你的**完整玩家 ID**，系统才会执行。输错了或关掉则不扣款不执行。

## 费用说明

| 费用 | 多少 | 什么时候扣 |
|------|------|-----------|
| 创建费 | $20 | 创建合同时立即扣 |
| 完成手续费 | 报酬的 5% | 合同完成时从报酬中扣 |

> 有 `contract.bypass.fee` 权限的玩家免创建费。权限由管理员分配。

## 常见问题

**钱去哪了？**
创建合同的时候，报酬就已经从你账户扣走，存在服务器托管账户。对方完成后才转给对方。你在中途取消能退回（按规则）。

**对方不干活怎么办？**
取消合同，发起争议，或者等合同过期。过期后钱自动退回。

**裁判不公平怎么办？**
发起争议，管理员会介入重新裁定。

**能同时接几个活？**
每人最多同时有 3 个进行中的合同（作为雇主或雇员）。
